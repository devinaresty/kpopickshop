# KPopick - K-Pop E-Commerce Platform

Platform e-commerce untuk penjualan produk K-Pop dengan sistem pembayaran terintegrasi Xendit, role-based access control, dan admin dashboard.

## 🏗️ Tech Stack

### Backend
- **Framework**: NestJS v6.19.2
- **Runtime**: Node.js
- **Database**: PostgreSQL + Prisma ORM
- **Authentication**: JWT + Passport.js
- **Authorization**: Role-Based Access Control (RBAC)
- **Payment Gateway**: Xendit
- **Storage**: Minio S3
- **API Docs**: Swagger/OpenAPI
- **API Port**: 3000

### Frontend
- **Framework**: Vue.js 3
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **HTTP Client**: Axios
- **State Management**: Pinia
- **Router**: Vue Router

### Database
- **PostgreSQL** (development: localhost:5432)
- **Database**: kpk_db

## 📋 Prerequisites

- Node.js v18+
- npm/yarn
- PostgreSQL 13+
- Git
